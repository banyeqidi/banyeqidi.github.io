搭建个人Hexo博客

Posted on 2021-03-06 Edited on 2021-05-03 In categories

1.所需要下载的应用：(Nodejs,Git,Sublime）

Nodejs下载地址：Nodejs

Git下载地址:Git

Sublime下载地址：Sublime

下载完之后win+R打开 输入cmd

输入：node -v 可查看node的版本

2.注册一个Github账号

Github地址：Github

打开仓库新建一个仓库

3.打开Git Bash，配置SSH

点击鼠标右键，打开Git Bash Here

输入以下命令：ssh-keygen -t rsa -C”你的Github邮箱”

一直按空格即可（或者Y +空格）

出现一下内容既成功

下一步，打开Github

点击New SSH Key

打开C盘搜索id_rsa.pub这个文本打开它

将其中的SSH密钥复制下来

粘贴到Github的那个Key下面空白处（title可不填）

4.hexo的本地使用

新建一个文件夹（blog）

鼠标移到blog文件夹上点击右键，打开Git Bash

输入下面命令下载一个插件：npm install hexo-cli -g

下一步，win+R打开cmd，找到blog这个目录输入hexo init

下一步，输入:npm install

再下一步,输入:hexo s

打开浏览器进入：http://localhost:4000/

当看到Hexo网址成功打开既hexo搭建成功

5.用sublime打开blog文件中的_config

1.修改url的内容为：https://你的用户名(username).github.io(你创建仓库时的地址)

2.将deploy下的内容修改为如图所示

3.最后用DOS打开blog文件夹（或者用Git Bash输入以下的命令）

输入hexo clean

hexo g

hexo d

完成发布

最后登录你的hexo地址：你的用户名（username）.github.iio(仓库地址)

看到网址既成功搭建个人博客
